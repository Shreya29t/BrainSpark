# 🧠 BrainSpark – AI Flashcard & Quiz Generator

BrainSpark is an AI-powered learning platform that helps students generate smart flashcards and interactive quizzes for better learning and revision.

## 🚀 Features

* 📚 AI Flashcard Generation
* 📝 Interactive Quiz Generator
* 🎯 Instant Answer Feedback
* 🌙 Modern & Responsive UI
* 🔐 Authentication Support
* ⚡ Fast Performance with Next.js

## 🛠️ Tech Stack

**Frontend:**

* Next.js
* TypeScript
* CSS

**Backend:**

* Python

**Database / Services:**

* Firebase

BrainSpark/

## 💡 Purpose

This project was built to make studying smarter and more interactive by automatically generating flashcards and quizzes for users.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Shreya29t/BrainSpark.git
```

Go to the project folder:

```bash
cd BrainSpark
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
